# PeterPark 🎢

PeterPark é uma plataforma interativa de mini-jogos educativos desenvolvida com **React**, **TypeScript** e **Vite**.

## 🚀 Como Rodar o Projeto

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

## 🎮 Jogos Disponíveis
- **PeterRun:** Uma aventura de corrida com o coelho Peter.
- **Jogos de Habilidade:** Bubble Pop, Whack-a-Mole, Jogo da Memória.
- **Educativos:** Números, Formas e Música.
- **Criativos:** Jogo de Massinha e Alimentação.

## 🛠️ Estrutura do Projeto
- `src/games`: Lógica principal de jogos complexos.
- `src/views`: Telas dos mini-jogos.
- `src/components`: Componentes compartilhados.
- `public/sprites`: Animações e ativos visuais.
